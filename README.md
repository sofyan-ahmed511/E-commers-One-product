# LUMORA - Premium Modern Clothing E-Commerce

LUMORA is an ultra-modern, minimalist, high-quality contemporary clothing e-commerce platform designed for the future of everyday wear. It features a seamless user experience with smooth animations, a clean aesthetic, and a fully responsive design.

## Project Description

LUMORA was founded on a simple principle: clothing should empower the wearer through uncompromising quality, minimalist aesthetics, and functional design. This project serves as the digital storefront for the brand, offering a curated selection of outerwear, tops, bottoms, and accessories.

The platform is built with a focus on performance, accessibility, and visual elegance. It includes features such as:
- **Smooth Page Transitions & Animations:** Powered by Framer Motion for a fluid user experience.
- **Custom Preloader & Cursor:** A sophisticated entrance animation and interactive cursor that sets the tone for the brand.
- **Dynamic Navbar:** A persistent, elliptical navbar that seamlessly integrates with hero sections and adapts to scrolling.
- **Interactive Product Galleries:** Parallax scrolling effects and hover states for product discovery.
- **Global State Management:** Utilizing Zustand for a persistent shopping cart and UI state.
- **Responsive Layout:** Optimized for all screen sizes, from mobile devices to large desktop monitors.
- **High Performance:** Optimized images, static generation where applicable, and fast loading times.
- **Complete User Flow:** From product discovery to a seamless checkout experience.

## Skills & Technologies Used

This project was built using a modern frontend stack:

- **Framework:** [Next.js 15](https://nextjs.org/) (App Router)
- **Library:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://motion.dev/) (via `motion/react`)
- **State Management:** [Zustand](https://zustand-demo.pmnd.rs/) (with persistence middleware)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Smooth Scrolling:** [Lenis](https://lenis.studiofreight.com/)
- **Language:** [TypeScript](https://www.typescriptlang.org/)

## Getting Started

To run this project locally, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd lumora-ecommerce
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) to view the application.

## Deployment

This project is optimized for deployment on platforms like Vercel, Netlify, or Google Cloud Run. Ensure that all environment variables (if any) are properly configured in your deployment environment.

To build the project for production:
```bash
npm run build
```

To start the production server:
```bash
npm run start
```
